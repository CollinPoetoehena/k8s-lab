## Structure of Design Documents
The HLD and LLD describe the overall design and features, while the folders in the design directory cover specific design decisions related to different aspects of the system, such as services, infrastructure, storage, security, tools, etc., all using ADRs (Architecture Decision Records) to document the decisions made, the alternatives considered, and the rationale behind each decision.
